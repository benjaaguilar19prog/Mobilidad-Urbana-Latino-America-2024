# Mobilidad-Urbana-Latino-America-2024

### "Movilidad urbana y productividad económica" | Proyecto Sprint-5 | TripleTen

Este repositorio contiene datos de trafico (en tiempo real), PIB per capita, desempleo y población por las grandes capitales en el mundo.

El Proyecto forma parte del curso de formacion Data Analyst de TripleTen. Sprint 5.

### INFORMACION DE LOS DATOS :

1. Urban Mobility - PIB - Proyect 5.ipynb

2. Urban Mobility - PIB - Proyect 5.csv

	Reporte final del analisis.

### PARA VERIFICAR EL ANALISIS : 

1. abrir el archivo .ipynb
2. archivo .csv (informe analisis)

### CONTEXTO DEL ANALISIS :

Entender como la movilidad urbana se relaciona con el PIB per capita y con las tasas de desempleo.
Consideraremos variables como las que se enlistan :
1. Tiempo en congestion.
2. Tiempo de retraso.
3. Ciudad.
4. Pais.
5. Año.
6. GDP. (Producto Interno Bruto en USD)
7. Tasa desempleo.
8. Indicadores de contaminacion.
9. Población.

Fuentes primarias de datos :
1. tomtom_traffic.csv : Datos sobre congestión vehicular y condiciones de tráfico. Informacion proporcionada por una empresa global de geolocalizacion.
2. oecd_city_economy.csv : Indicadores económicos y ambientales por ciudad. Informacion	proporcionada por la OECD (Organizacion para la Cooperacion y Desarrollo Economico, siglas en ingles)
### Archivos no incluidos en el repositorio ###

Trabajamos con los 2 Datasets para poder relacionar variables, p/ej: alta congestion - baja productividad, movilidad eficiente - economia fuerte, baja productividad - posibilidad inversion en infraestructura. Asi poder entregar un reporte para entender cómo la movilidad urbana se relaciona con la productividad económica en las principales ciudades de Latino America para el año 2024

### OBJETIVO DEL ANALISIS :

Aplicar limpieza, estandarización y validación de tipos de datos.
Limpiar, unir y analizar ambas bases para relacionar variables.
Filtrar y enfocar el análisis en el año 2024.
Calcular indicadores agregados (por ciudad–año).
Realizar análisis exploratorios y visuales.

Identificar ciudades Latino Americanas necesitan invertir en infraestructura de transporte para aumentar la productividad y el bienestar de la población.

Exportamos 1 dataset final (archivo .csv) completamente depurado con las conclusiones finales de analisis.
